ChatBois Android app

An android app for chatting with friends. This android app uses firebase extensively. The user logs in using his google account and can chat in the general room which for now consists of all the users ujsing the app. The app also has a notification feature hence the user will be notified when a message is sent on the chat. This app uses database on firebase for it's storage of messages.

Last updated in April 2020

TechnoMedic Android app

An android app which uses REST API from a medical database. Here any user can search for symptoms that he /she is currently facing. The user can select those symptoms , choose his gender and age and tap diagnose. This will give the user a list of diseases or medical conditions he might be facing with an accuracy score. The details also include the medical name, How the medical condition is cured and a short description of the ailment.

Last updated in April 2020

WeatherWizard Android app

One of the first android apps I built. It is pretty simple which uses a ready made Rest APi from a weather source. User can tap the GPS icon on the app to view the weather in his current location and also search for any city in the world to view it's current temperature. Temperature stats include:

  • Minimum Temperature
  • Maximum Temperature
  • Sunrise time
  • Sunset time
  • Wind Speed
  • Weather Condition
  • Pressure
  • Humidity

Last updated in June 2019
